5. **Link rot:** if a source link dies, check the Wayback Machine, then the Master Catalog for alternate access points; update the catalog — it's the single source of truth for access.

## Quality rules (unchanged, non-negotiable)

- Summaries are original prose; never paste copyrighted commentary text.
- Quotes ≤25 words, attributed, linked, spot-checked before print/publication.
- Every commentary entry links its source — an unlinked entry is a bug.
- The synthesis line maps convergence/divergence and asks a question; it never picks a winner.
